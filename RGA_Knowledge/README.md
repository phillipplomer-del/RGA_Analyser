# RGA Knowledge Database

This directory contains raw data files for RGA analysis: isotope data, RSF values, gas properties, and mass-to-gas mappings.

---

## Data Files

### [isotopeData.md](./isotopeData.md)
Raw isotope abundance data from CIAAW (Commission on Isotopic Abundances and Atomic Weights).

### [mass_to_gas_mapping.md](./mass_to_gas_mapping.md)
Mass-to-gas reference table for common gases detected by RGA.

### [RSF_Values.md](./RSF_Values.md)
Relative Sensitivity Factors (RSF) for quantitative RGA analysis.

### [gasDatabase.md](./gasDatabase.md)
Comprehensive gas properties database (molecular weight, ionization potential, etc.).
